# Regulus
Regulus Debris Guardian - это веб-приложение, которое в реальном времени по актуальным данным о космических объектах из официальных источников (Space-Track.org), прогнозирует траектории, автоматически выявляет потенциальные столкновения и генерирует готовые к исполнению планы манёвров уклонения для спутников.
