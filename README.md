# Regulus Debris Guardian

**Полнофункциональное веб-приложение для мониторинга и управления рисками столкновений космических объектов в реальном времени.**

Ccылка на сайт (задеплоен на Vercel) --> **regulusai.vercel.app**

## О проекте

Regulus Debris Guardian — это инструмент, который в реальном времени загружает актуальные орбитальные данные из официального источника **Space-Track.org**, прогнозирует траектории объектов, выявляет потенциальные столкновения и автоматически генерирует готовые рекомендации по манёврам уклонения с помощью ИИ (Mistral-7B).

Проект разработан для участия в международном конкурсе **AEROO Space AI Competition** и демонстрирует применение искусственного интеллекта в области космической ситуационной осведомлённости (Space Situational Awareness).

### Основные возможности
- Автоматическое обновление данных (TLE/OMM, CDM) каждые сутки
- Прогнозирование траекторий на 7 суток вперёд (LSTM-модель)
- Выявление рисков столкновений с расчётом вероятности и минимального расстояния
- Генерация текстовых планов манёвров уклонения (Δv, направление, время)
- Интерактивная 3D-визуализация орбит с цветовой индикацией рисков
- Мгновенные уведомления о критических рисках через WebSocket

### Демонстрация данных

Все необходимые файлы (satcat.csv, gp_recent.tle, cdm_public.csv, gp_history_7d.csv и др.) размещены во временном облачном хранилище исключительно для членов жюри:

### Технологический стек
- Frontend: Next.js 14 (App Router) + React 19 + TypeScript + Tailwind CSS + shadcn/ui + React Three Fiber + Three.js
- Backend / API: Next.js API Routes
- База данных: Neon PostgreSQL (serverless)
- AI: Mistral-7B (API) + внешний Python-сервис с LSTM-моделью
- Орбитальные расчёты: satellite.js (клиент) + skyfield/poliastro (сервер)
- Деплой: Vercel (frontend + backend)
- Аутентификация: JWT + bcryptjs
